# Resources for Shotcut

[Shotcut](www.shotcut.org) is a great free, open source, cross-platform video editor.
 

## Filter sets
Filter set must be copied into the filter-set folder in the Shotcut data directory.
The data directory is found by opening Shotcut and select 

**Settings -> App Data Directory -> Show**

### Fonts
The following fonts are used by some of the filter sets.

- [Montserrat](https://www.fontsquirrel.com/fonts/montserrat)
- [Spicy Rice](https://www.fontsquirrel.com/fonts/spicy-rice)
- [American Captain](https://www.dafont.com/american-captain.font)
